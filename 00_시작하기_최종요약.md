## ✨ 핵심 기능

### 1️⃣ 실제 네이버 플레이스 형식 지원
- ✅ 긴 소개글 (1000자 이상) 자동 분석
- ✅ 진료과목, 메뉴 상세 설명 모두 처리
- ✅ 복잡한 구조도 OK

### 2️⃣ 똑똑한 AI 분석
```
입력: "매주 월~목 밤 8시 30분까지 야간진료..."
     "토/일요일 진료"
     "비대면 상담 후 약처방 가능"

AI 감지:
  ⭐ 야간진료
  ⭐ 주말진료
  ⭐ 비대면가능
```

### 3️⃣ 허위정보 자동 차단
```
입력: "한강 도보 10분"
차단: ❌ "한강뷰", "리버뷰" (직접 인접 아님)
허용: ✅ "한강 가까운", "한강공원 근처"
```

### 4️⃣ 업종별 특화 키워드
- **한의원**: 체질맞춤, 약침잘하는, 탈모잘보는
- **병원**: 실력파원장, 꼼꼼진료, 양심의료
- **식당**: 단골맛집, 재방문각, 가성비갑
- **카페**: 감성카페, 작업하기좋은, 인스타감성
